# 새로 구현된 10가지 기능 요약

## ✅ 구현 완료된 기능 목록

### 1. 가독성 점수 계산기 (`readability-score`)
- **위치**: `/tools/readability-score`
- **카테고리**: 글 길이/분량
- **기능**: 
  - Flesch Reading Ease 공식을 기반으로 가독성 점수 계산 (0-100점)
  - 읽기 수준 평가 (매우 쉬움, 쉬움, 보통, 어려움, 매우 어려움)
  - 개선 제안 자동 제공
- **사용 방법**: 텍스트를 입력하면 자동으로 가독성 점수와 개선 제안이 표시됩니다.

### 2. 키워드 밀도 분석기 (`keyword-density`)
- **위치**: `/tools/keyword-density`
- **카테고리**: 메타·SEO
- **기능**:
  - 특정 키워드의 출현 횟수 및 밀도 계산
  - 상위 20개 키워드 자동 추출 및 빈도 분석
  - SEO 최적화 권장 밀도 (1-3%) 표시
- **사용 방법**: 분석할 키워드를 입력하면 해당 키워드의 밀도와 전체 키워드 목록이 표시됩니다.

### 3. 텍스트 비교 도구 (`text-comparison`)
- **위치**: `/tools/text-comparison`
- **카테고리**: 분석
- **기능**:
  - 두 텍스트 간 유사도 계산 (Jaccard 유사도 기반)
  - 차이점 자동 추출 (추가, 삭제, 변경)
  - 시각적 차이점 표시
- **사용 방법**: 원본 텍스트와 비교할 텍스트를 입력하면 유사도와 차이점이 표시됩니다.

### 4. 단어 빈도 분석기 (`word-frequency`)
- **위치**: `/tools/word-frequency`
- **카테고리**: 분석
- **기능**:
  - 텍스트 내 단어 빈도 분석
  - 상위 30개 단어 자동 추출
  - 빈도수와 비율(%) 표시
  - 진행 바로 시각화
- **사용 방법**: 텍스트를 입력하면 단어 빈도가 자동으로 분석되어 표시됩니다.

### 5. 문장 난이도 분석기 (`sentence-difficulty`)
- **위치**: `/tools/sentence-difficulty`
- **카테고리**: 분석
- **기능**:
  - 문장별 난이도 분석 (쉬움, 보통, 어려움)
  - 평균 문장 길이 및 단어 수 계산
  - 전체 텍스트 난이도 평가
  - 문장별 상세 분석 (최대 20개)
- **사용 방법**: 텍스트를 입력하면 문장별 난이도와 전체 통계가 표시됩니다.

### 6. 요약문 생성기 (`text-summarizer`)
- **위치**: `/tools/text-summarizer`
- **카테고리**: 분석
- **기능**:
  - 긴 텍스트를 핵심 내용으로 요약
  - 첫 문장, 긴 문장, 마지막 문장을 조합하여 요약 생성
  - 요약문 복사 기능
  - 미리보기에 적용 기능
- **사용 방법**: 텍스트를 입력하면 자동으로 요약문이 생성됩니다.

### 7. 키워드 클라우드 생성기 (`keyword-cloud`)
- **위치**: `/tools/keyword-cloud`
- **카테고리**: 분석
- **기능**:
  - 텍스트에서 상위 20개 키워드 추출
  - 빈도에 따라 글자 크기 자동 조절
  - 클라우드 형태로 시각화
  - 호버 시 상세 정보 표시
- **사용 방법**: 텍스트를 입력하면 키워드 클라우드가 자동으로 생성됩니다.

### 8. 텍스트 통계 대시보드 (`text-statistics`)
- **위치**: `/tools/text-statistics`
- **카테고리**: 분석
- **기능**:
  - 종합 텍스트 통계 제공 (글자수, 단어수, 문장 수, 문단 수)
  - 평균 통계 계산 (문장당 평균 단어 수, 단어당 평균 글자 수)
  - 가독성 점수 통합 표시
  - 카드 형태로 시각화
- **사용 방법**: 텍스트를 입력하면 모든 통계가 한눈에 표시됩니다.

### 9. CSV/TSV 정리기 (`csv-cleaner`)
- **위치**: `/tools/csv-cleaner`
- **카테고리**: 추출·정리
- **기능**:
  - CSV/TSV 포맷 선택 (쉼표 또는 탭 구분)
  - 셀 앞뒤 공백 자동 제거
  - 빈 행 제거
  - 중복 행 제거
- **사용 방법**: CSV/TSV 데이터를 입력하고 옵션을 선택하면 정리된 데이터가 표시됩니다.

### 10. JSON 포맷터 (`json-formatter`)
- **위치**: `/tools/json-formatter`
- **카테고리**: 변환
- **기능**:
  - JSON 데이터 포맷팅 (들여쓰기 조절 가능)
  - JSON 압축 (한 줄로 변환)
  - JSON 검증 및 에러 표시
  - 들여쓰기 공백 수 조절 (0-8)
- **사용 방법**: JSON 데이터를 입력하면 자동으로 포맷팅되거나 압축됩니다.

## 🎯 주요 기능 특징

### 분석 도구 통합
- 모든 분석 도구를 `AnalysisTools` 컴포넌트로 통합하여 코드 재사용성 향상
- 각 도구는 `toolId`로 구분되어 조건부 렌더링

### 사용자 친화적 UI
- 카드, 배지, 진행 바 등 shadcn/ui 컴포넌트 활용
- 반응형 디자인 지원
- 명확한 레이블과 설명 제공

## 🚀 사용 예시

### 가독성 점수 계산기
```
입력: "이것은 테스트 문장입니다. 매우 간단한 문장입니다."
출력: 가독성 점수 85점, "매우 쉬움" 수준
```

### 키워드 밀도 분석기
```
입력: "텍스트 분석 도구는 매우 유용합니다. 텍스트 분석을 통해 인사이트를 얻을 수 있습니다."
키워드: "텍스트"
출력: 출현 횟수 2회, 밀도 8.7%
```

### 텍스트 비교 도구
```
원본: "안녕하세요. 반갑습니다."
비교: "안녕하세요. 만나서 반갑습니다."
출력: 유사도 75%, 차이점: "만나서" 추가됨
```
